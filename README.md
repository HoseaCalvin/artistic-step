# Artistic Step
This is a shoe store application written in Java that uses JavaFX framework and XAMPP MySQL Database Engine.

## Disclaimer
The picture displayed on the bottom right pane is randomized, not retrieved from the actual database.

## Prerequisite
1. Please install XAMPP first to be able to run the database engine.
2. Create a database named "artistic_step".
3. Create two tables - "shop_shoes" and "user_shoes".
4. Add attributes to the "shop_shoes" table.
![image](https://github.com/user-attachments/assets/998c524a-d547-44e4-9f99-4dc127aec8df)

5. Add attributes to the "user_shoes" table.
![image](https://github.com/user-attachments/assets/60921b42-c875-4c89-884f-438b4d168d72)

6. Finally, ensure XAMPP is online during the use of the application.
